# Xiaomi-Explorer

Interactive invitation prototype for Xiaomi Explorers: The New Wave of AI.

## Run locally

Camera access requires https or localhost. Use a local server:

```
python -m http.server 8000
```

Then open http://localhost:8000

## Config

Edit config.js to update copy, theme CSS variables, page backgrounds, KV slide
content, particle behavior, mask settings, gesture navigation, and KV images.

## Files

- index.html
- styles.css
- config.js
- app.js
- PROMPT.md (MiMo prompt to recreate the project)
